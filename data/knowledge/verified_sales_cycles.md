## Key Corrections from Previous Analysis

The earlier analysis showed inflated cycle times (400+ days average) because it counted:
- Post-sale support emails
- Trade show invitations years later
- Internal system emails
- Unrelated follow-up correspondence

- Standard deals: **3-6 months**

## True Sales Cycle Patterns

### Fast Closers (1-3 months)
- **Trade show leads** (K-Show, Elmia): Batelaan, JoPlast
- **Partner referrals**: DutchTides (via FRIMO)
- **Urgent replacements**: Customer has broken machine

### Medium Cycle (3-6 months)
- Standard inquiry → quote → evaluation → order
- Typical for deals €50K-€150K

### Long Cycle (12-18 months)
- Multiple stakeholders
- Budget cycle alignment
- Extensive technical evaluation (German market)
- New market entry (building trust)

## What Accelerates Deals

1. **Trade show meeting** - Face-to-face builds trust immediately
2. **Factory visit** - Seeing machine in operation converts
3. **Partner introduction** - FRIMO, Ridat referrals
4. **Urgent need** - Broken machine, new project deadline
5. **Clear specifications** - Customer knows exactly what they need

## What Extends Cycles

1. **Multiple decision makers** - Each needs convincing
2. **Budget cycle** - Waiting for new fiscal year
3. **Technical evaluation** - Especially in Germany
4. **Competitive comparison** - Evaluating CMS, Illig
5. **Trust building** - New markets, no references nearby
