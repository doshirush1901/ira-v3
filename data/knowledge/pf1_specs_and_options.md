# PF1 Specs & Options — Clarifications

Use this so Ira and content use correct PF1 terminology and numbers.

## PF1-X-5028 (and large XL)

- **516 kW heater:** Top and bottom **combined** (not 516 kW top only). Total connected load 602 kW = 516 kW heater + 86 kW servo.
- **Sheet thickness:** 2–12 mm (same range as PF1-C for cut-sheet single-station).
- **Sheet loading:** Manual or with autoloader — both are options; quote as ordered.
- **Universal frames:** **Sheet size changeover system** (for quick size changeover), not just "for loading". Describe as changeover system when explaining to customers.

## PF1-C and PF1-X (single-station, cut-sheet)

- **Sheet thickness:** **2–12 mm** for both (not 2–6 mm only for PF1-C). Use 2–12 mm in specs and comparison tables unless a specific model brochure says otherwise.
- **Sheet loading:** Manual or with autoloader where the option exists (PF1-X: autoloader optional; PF1-C: typically manual, check model).
- **Universal frames (PF1-X):** Sheet size changeover system; optional.
